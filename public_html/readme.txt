Настройка:
    1) php composer.phar install
    2) В настройкай сервера указать директорию public как корневую. Можно взять за основу файл test.conf.example
    3) Импортировать базу даннных test7.sql
    4) В файле config/autoload/global.php указать креденшиалы к базе
    5) Проверить права на файлы
    6) Перейти на site.loc/ и проверить конекшин
